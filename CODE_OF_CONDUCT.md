# Code of Conduct

## Our Pledge

We are committed to making this project a welcoming, inclusive space for everyone — regardless of experience level, background, or identity.

## Our Standards

**Expected behaviour:**
- Be respectful and constructive in all feedback and discussions
- Welcome beginners — everyone starts somewhere
- Give credit when building on others' work
- Accept feedback gracefully

**Unacceptable behaviour:**
- Harassment, insults, or personal attacks
- Dismissing questions as "too basic"
- Plagiarising solutions and presenting them as original work

## Scope

This Code of Conduct applies to all project spaces: GitHub Issues, Pull Requests, Discussions, and any community channels.

## Enforcement

Violations may be reported by opening a GitHub Issue or contacting the maintainers directly. All reports will be reviewed and responded to promptly. Maintainers reserve the right to remove comments or ban contributors who violate this Code of Conduct.
